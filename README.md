# 🚀 Python Journey Tracker

Программа-трекер для контроля времени обучения Python с системой мотивации в стиле **Duolingo** и личным дневником достижений.

Проект создан для закрепления навыков Python и постепенно развивается вместе с моим обучением.

---

## 🛠️ Особенности и функции

- 🔥 **Система страйков (Duolingo-style)** — считает дни непрерывного обучения.
- ⏱️ **Учёт времени** — добавление времени обучения и отдыха.
- 📅 **Гибкое логирование** — можно добавить запись за **сегодня** или **вчера**.
- 📝 **Дневник успехов** — оценка продуктивности дня (от 1 до 5) и запись главных достижений.
- 📊 **Недельная аналитика** — подсчёт часов и просмотр статистики за последние 7 дней.
- 🎯 **Цель недели** — главная задача отображается в главном меню.
- 💾 **Автоматическое сохранение** всех данных в формате JSON.

---

## 📦 Архитектура проекта

Проект построен по модульной структуре.

- `src/tracker/` — основная логика программы.
- `src/main.py` — точка входа.
- `data/` — хранение пользовательских данных.
- `pyproject.toml` — настройки проекта и Poetry.

---

## 🚀 Запуск проекта

Убедитесь, что установлен **Poetry**.

```bash
poetry install
poetry run python src/main.py
```
