# Python Journey — Личный трекер обучения и дневник достижений

Программа для отслеживания времени изучения Python и ведения ежедневного дневника успехов. Помогает контролировать прогресс, не бросать учебу благодаря системе страйков и анализировать продуктивность в конце недели.

## ✨ Что нового в версии v2.0
- 🔥 **Система страйков (Duolingo-style)**: Счетчик дней непрерывной учебы прямо в главном меню. Не дайте своему огню погаснуть!
- 📝 **Личный дневник перед сном**: Отдельный пункт для записи главных побед за день и оценки продуктивности от 1 до 5.
- 📅 **Умные итоги недели**: Программа сама отсчитывает последние 7 дней, собирает все изученные темы, считает часы и выводит ваши достижения.
- 🎯 **Гибкое планирование**: Возможность ставить фокус-цель на следующую неделю курса или пропустить этот шаг.
- 🗑 **Продвинутое удаление**: Теперь можно выбрать любую запись из списка по номеру с обязательным подтверждением (Да/Нет), чтобы избежать случайной потери данных.
- 🧼 **Чистый интерфейс**: Консоль автоматически очищается после каждого действия, больше никакой «каши» из текста!

## Функции
- 📅 **Автоматическая дата**: Программа сама определяет сегодняшний день.
- ⏱ **Объединение записей**: Если добавить учебу несколько раз за день, минуты сложатся, а темы запишутся через запятую.
- 💾 **Надежное сохранение**: Все данные записываются в файл `journal.json`. При обновлении программы прошлые дни не стираются.

## Как запустить
1. Убедитесь, что у вас установлен Python.
2. Скачайте файлы проекта.
3. Дважды кликните по файлу `run.bat` (для Windows).
