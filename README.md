# Python Journey — Трекер обучения

Простая и удобная консольная программа для отслеживания времени изучения Python. Помогает контролировать прогресс, записывать изученные темы и вести статистику учебных дней и дней отдыха.

## Функции
- 📅 **Автоматическая дата**: Программа сама определяет сегодняшний день.
- ⏱ **Объединение записей**: Если добавить учебу несколько раз за день, минуты сложатся, а темы запишутся через запятую.
- 📊 **Полная статистика**: Подсчет общего времени в минутах и часах, а также количества учебных дней и дней отдыха.
- ❌ **Удаление записей**: Возможность легко стереть последнюю ошибочную запись.
- 💾 **Сохранение данных**: Все данные надежно сохраняются в файл `journal.json`.

## Как запустить
1. Убедитесь, что у вас установлен Python.
2. Скачайте файлы проекта.
3. Дважды кликните по файлу `run.bat` (для Windows).
